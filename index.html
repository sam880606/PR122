通用教學：手動多開與捷徑設定

1

建立捷徑 (Create Shortcut)

右鍵點選解壓縮後的 flashplayer...exe，選擇「傳送到」>「桌面 (建立捷徑)」。

2

設定基礎路徑 (Set Base Target)

右鍵捷徑 >「內容」。在「目標」欄位最後面空一格，貼上：

https://tw1mc.funmily.com/s/s7/GameLoader.swf?

* Windows 11 請務必確認有空格。

3

獲取登入參數 (Get User Session)

  • 前往官方網頁登錄帳號並進入伺服器。
  • 在灰色畫面按右鍵 > 「檢視網頁原始碼」。
  • 找到第 41 行左右的 user= 字串。
  • 複製直到 ' + 之前結束。
4

完成設定 (Finalize)

回到捷徑內容，將代碼貼在 GameLoader.swf? 後面。

範例：...exe" https://...GameLoader.swf?user=123...
